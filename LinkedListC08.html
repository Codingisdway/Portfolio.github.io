Linked List

Skip M and delete N nodes.

QUESTION :

Given a linked list and two integers M and N. Traverse the linked list such that you retain M nodes then delete next N nodes, continue the same till end of the linked list.

LOGIC :

Simple iterative approach is used. First we skip m elements and then we delete n elements.

CODE :

/* Linked list Node*/

class Node {

int data;

Node next;

Node(int data){this.data=data;}

}

/*Solution :*/

class Solution{

public Node skipMdelN(Node head,int m,int n){

if(head==null) return head; //base case
