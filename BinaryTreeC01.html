Binary Tree

Depth of a Binary Tree.

QUESTION :

Given a binary tree, the task is to find the height of the tree. The height of the tree is the number of edges in the tree from the root to the deepest node.

LOGIC :

We have used a simple recursion approach.

We will recursively do the DFS, till we reach the leaf nodes.

Then return the maximum value of left or right node.

CODE :

/* Binary Tree Node*/

public class TreeNode{

int data;

TreeNode left=null;

TreeNode right=null;

TreeNode(int data){this.data=data;}

}

/*Solution :*/

class Solution{

public int height(TreeNode root){

if(root==null) return 0; //base case

int leftHeight=height(root.left);

int rightHeight=height(root.right);

return (Math.max(leftHeight,rightHeight)+1)

}

}

ANALYSIS :

Time : O(n)

Space : O(n)
