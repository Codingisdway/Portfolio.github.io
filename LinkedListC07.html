Linked List

Detect a cycle.

QUESTION :

Given head, the head of a linked list, determine if the linked list has a cycle in it. There is a cycle in a linked list if there is some node in the list that can be reached again by continuously following the next pointer. Internally, pos is used to denote the index of the node that tail's next pointer is connected to. Note that pos is not passed as a parameter. Return true if there is a cycle in the linked list. Otherwise, return false.

LOGIC :

We have used Hare-Tortoise algorithm. A Hare-Tortoise algorithm is a pointer algorithm that uses only two pointers, moving through the sequence at different speeds, one moving twice as fast as the other one.

CODE :

/* Linked list Node*/

class Node {

int data;

Node next;

Node(int data){this.data=data;}

}

/*Solution :*/

class Solution{

public boolean detectLoop(Node head){ // Floyd cycle detection algorithm

if(head==null) return head; //base case

Node fast = head, slow = head;

while(fast!=null && fast.next!=null ){

fast=fast.next.next;

slow=slow.next;

if(fast==slow) return true;

}

return false;

}

}

ANALYSIS :

Time : O(n)

Space : O(1)
