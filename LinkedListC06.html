Linked List

Remove duplicates.

QUESTION :

Given the head of a linked list and an integer val, remove all the duplicate nodes of the linked list, and return the new head.

LOGIC :

Simple iterative approach is used, if the val is equal to key then delete the element.

CODE :

/* Linked list Node*/

class Node {

int val;

Node next;

Node(int data){this.val=data;}

}

/*Solution :*/

class Solution{

public Node removeDuplicates(Node head){

if(head==null) return head; //base case

Node dummy=head;

while(dummy != null) {

if(dummy.val==dummy.next.val){

dummy.next=dummy.next.next;

continue;

}

dummy=dummy.next;

}

return head;

}

}

ANALYSIS :

Time : O(n)

Space : O(1)
