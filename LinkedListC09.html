Linked List

Link two linklists.

QUESTION :

You are given two linked lists: list1 and list2 of sizes n and m respectively.

Remove list1's nodes from the ath node to the bth node, and put list2 in their place. Build the result list and return its head.

LOGIC :

First we traverse the first list till a. Then we break from the first list and point to the second list. After we finist traversing the entire second list we link back to the first.

CODE :

/* Linked list Node*/

class ListNode {

int data;

ListNode next;

ListNode(int data){this.data=data;}

}

/*Solution :*/

class Solution{

public ListNode mergeInBetween(ListNode list1, int a, int b, ListNode list2) {

for (int i = 0; i < b; ++i, end = end.next) { // locate b upon termination of the loop.

if (i == a - 1) start = end; // locate the node right before a.

}

start.next = list2;

// connect the node right before a to the head of list2, hence cut off a from list1.

while (list2.next != null) list2 = list2.next; // traverse till the end of list2.

list2.next = end.next; // connect end of list2 to the node right after b.

return list1;

}

}

ANALYSIS :

Time : O(n)

Space : O(1)
