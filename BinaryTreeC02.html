QUESTION :

Write a function to determine if two trees are identical or not: Two trees are identical when they have the same data and the arrangement of data is also the same.

LOGIC :

To identify if two trees are identical, we need to traverse both trees simultaneously, and while traversing we need to compare data and children of the trees.
